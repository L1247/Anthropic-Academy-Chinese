## 📖 課程簡介

**如果你只能選一門課，選這門。**

Claude 101 是認識 Claude 最快速的方式。課程帶你了解 Claude 的核心功能、實用工作流程和最佳實踐，讓你在最短時間內就能把 Claude 用在日常工作中。

無論你有沒有用過其他 AI 工具（如 ChatGPT），這門課都能幫你快速掌握 Claude 的特色和使用方式。

## 🎯 學習目標

完成本課程後，你將能夠：

- 了解 Claude 的**核心能力**和適合的使用場景
- 使用 Claude 完成**日常工作任務**：寫信、整理資料、分析文件
- 掌握**有效提示**的基本原則
- 了解如何進一步深入學習其他進階課程

## 📋 課程大綱

### 💬 單元一：認識 Claude
- Claude 是什麼？和其他 AI 助手有什麼不同？
- Claude 的核心功能概覽
- 如何登入和開始使用 claude.ai

### 💼 單元二：日常工作應用
- **寫作輔助**：草稿撰寫、文章改寫、摘要生成

- **初稿產出**：電子郵件、報告、企劃書

### ✍️ 單元三：有效提示技巧
- 清晰描述你的需求

### ⚡ 單元四：Claude 的特色功能
- 長上下文視窗：一次處理大量文字

## 📝 重點筆記

### ⚖️ Claude 和 ChatGPT 的主要差異

| 特性 | Claude | ChatGPT |
|------|--------|---------|
| 上下文長度 | 更長（適合長文件） | 較短 |
| 寫作風格 | 更自然、更少廢話 | 有時較制式 |
| 安全性 | Constitutional AI 設計 | RLHF 設計 |
| 程式碼能力 | 強 | 強 |
| 資料截止 | 有，但可上傳文件補充 | 有，有網路搜尋 |

### 🎯 最有效的 Claude 使用場景

**高效場景（強烈推薦）：**
- 長文摘要與重點提取
- 寫作草稿與改寫優化
- 程式碼說明與偵錯
- 複雜概念的解釋

**中等效果（需要驗證）：**
- 數據分析（複雜計算需自行驗證）
- 資訊查詢（注意知識截止日）

**不建議場景：**
- 需要即時資訊的任務（使用搜尋引擎）
- 重要決策的唯一依據

### 📧 提示範本：電子郵件寫作

```
請幫我寫一封電子郵件，內容如下：
- 收件人：[職稱/稱謂]
- 目的：[你想達成的目標]
- 語氣：[正式/輕鬆/感謝]
- 需要包含的重點：[列出 2-3 個重點]
- 長度：[短/中/長]
```
