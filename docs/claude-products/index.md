## ⚖️ 三款產品的差異

| 產品 | 使用方式 | 最適合 |
|------|---------|--------|
| **Claude（claude.ai）** | 網頁 / 手機 App | 一般對話、寫作、分析 |
| **Claude Code** | 命令列工具（CLI） | 開發者：程式碼生成、偵錯 |
| **Claude Cowork** | 桌面應用程式 | 處理本地檔案、多步驟任務 |

## 🗺️ 建議學習順序

1. **Claude 101** → 了解 Claude 的基本能力和使用方式
2. **Claude Cowork 入門** → 學習直接操作本地檔案
3. **Claude Code 101** → 如果你是開發者，進一步學習程式碼輔助工具
