## 🐙 GitHub 開源課程

### 📚 anthropics/courses

**連結：** [github.com/anthropics/courses](https://github.com/anthropics/courses)

Anthropic 官方 GitHub 課程倉庫，提供 5 組互動式 Jupyter Notebooks：

| 模組 | 主題 | 適合對象 |
|------|------|---------|
| 1. API 基礎 | 認證、基本請求、回應處理 | API 入門者 |
| 2. Prompt 工程 | 提示技巧、鏈式推理、XML 標籤 | 所有開發者 |
| 3. 工具使用（Tool Use） | 定義工具、解析呼叫、串連工具 | 中級開發者 |
| 4. RAG 實作 | 文字分塊、Embeddings、向量搜尋 | 中-高級開發者 |
| 5. Agent 架構 | 多步驟代理、路由、平行化 | 高級開發者 |

> **使用方式：** Clone 倉庫後，在 Jupyter 或 Google Colab 中執行 `.ipynb` 檔案即可。

## 👥 Claude 社群資源

### 🏠 官方社群

- **Anthropic Discord** — 開發者社群、問答、公告
- **Claude.ai 官網** — 直接使用 Claude 的網頁介面

### 🛠️ 實作練習建議

完成課程後，可以透過以下方式鞏固所學：

1. **Claude 101** → 實作：用 Claude 完成一週的真實工作任務，記錄成效
2. **Building with Claude API** → 實作：建立一個簡單的 CLI 工具，整合 Claude API
3. **MCP 入門** → 實作：建立一個 MCP Server，讓 Claude 能讀取你的本地資料夾
4. **Claude Code** → 實作：用 Claude Code 完成一個真實的 GitHub Issue 修復
5. **Agent Skills** → 實作：為你最常做的重複任務建立一個 Skill
