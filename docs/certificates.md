# 🎓 官方證書

Anthropic Academy 的所有課程皆**完全免費**，部分課程完成後還能獲得官方電子結業證書（Certificate of Completion）。證書可直接加入 LinkedIn 個人檔案或下載儲存，是展示 AI 學習成果的最佳證明。

::: tip 🔑 重點摘要
- 不需要考試，完課即自動頒發
- 完全免費，只需免費的 Skilljar 帳號
- 可加入 LinkedIn 個人檔案或履歷
:::

## 📋 如何取得證書

1. **免費註冊帳號** — 前往 [Anthropic Academy](https://anthropic.skilljar.com/) 註冊 Skilljar 帳號（不需要 Anthropic 帳號）
2. **選擇課程** — 挑選想學習的課程並開始上課
3. **完成所有單元** — 觀看影片、閱讀內容，部分課程需完成期末評量
4. **自動頒發證書** — 完成後系統會自動頒發電子結業證書

## ⚖️ 結業證書 vs CCA 認證

Anthropic 目前有兩種不同的「證書」，請注意區分：

| | 結業證書 | CCA 認證 |
|--|---------|---------|
| **全名** | Certificate of Completion | Claude Certified Architect |
| **費用** | 免費 | 付費考試 |
| **取得方式** | 完課即得，不需考試 | 需通過監考考試 |
| **報名資格** | 任何人皆可 | 目前僅限 Claude Partner Network 成員 |
| **性質** | 學習完成證明 | 正式技術資格認證 |
| **推出時間** | 隨課程上線 | 2026 年 3 月 12 日 |

::: warning ⚠️ 注意
CCA（Claude Certified Architect）認證目前僅開放給 Claude Partner Network 成員報考，一般使用者尚無法報名。對大多數學習者而言，完課取得結業證書已足夠展示學習成果。
:::

## ❓ 常見問題

::: details 需要 Anthropic 帳號嗎？
不需要。只需註冊免費的 Skilljar 帳號即可報名所有課程，與 Anthropic Console 帳號是獨立的系統。
:::

::: details 證書有期限嗎？
目前 Anthropic Academy 的結業證書沒有標示有效期限。但由於 AI 領域發展快速，課程內容可能會更新，建議定期回顧最新課程。
:::

::: details 如何將證書加入 LinkedIn？
完課後可在 Skilljar 平台下載電子證書，然後在 LinkedIn 個人檔案的「證照」區塊中手動新增，填入課程名稱、發證機構（Anthropic）及完課日期。
:::

::: details 課程內容是英文的嗎？
是的，Anthropic Academy 官方課程內容為英文。本網站提供的是繁體中文學習指南，幫助你理解課程重點與學習脈絡。
:::
