 1. 新增課程時：呼叫 doc-writer agent 撰寫頁面，完成後用 code-reviewer 審查
  2. 建置有問題時：直接呼叫 debugger agent

  新增課程時：
  @content-translator 翻譯課程

    → @practice-builder 建立練習頁
    → @git-commit 提交
